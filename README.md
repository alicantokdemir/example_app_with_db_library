# App Layout
![App Layout](./App_Layout.png)

# Requirements
node.js
web browser (latest firefox or chrome preferred)
# To install
run `npm i`
# To build
run `npm run build`
# To test (library)
run `npm run test`
# To serve
run `npm run serve` and navigate to `http://localhost:8080`
